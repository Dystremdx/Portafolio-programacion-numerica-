Graficador con Restricciones

Programa para graficar funciones lineales con restricciones y encontrar regiones factibles en problemas de optimización y programación lineal.

Programación Lineal

Descripción

Graficador avanzado que visualiza restricciones lineales y calcula automáticamente la región factible en problemas de optimización. Ideal para análisis de programación lineal y problemas con múltiples restricciones.

Librerías Utilizadas
NumPy Matplotlib SciPy itertools
Estructura de Restricciones
(a, b, c, tipo)
a*x + b*y ≤ c → tipo = "<="
a*x + b*y ≥ c → tipo = ">="

import numpy as np
import matplotlib.pyplot as plt
from itertools import combinations
from scipy.spatial import ConvexHull

def graficar_restricciones(restricciones, limites=(0, 20, 0, 20)):
    """
    Grafica restricciones lineales y su región factible.

    restricciones : lista de tuplas (a, b, c, tipo)
        a*x + b*y <= c ---> tipo = "<="
        a*x + b*y >= c ---> tipo = ">="
    limites : tupla (xmin, xmax, ymin, ymax)
    """
    xmin, xmax, ymin, ymax = limites
    x = np.linspace(xmin, xmax, 400)
    
    plt.figure(figsize=(7, 7))

    # Dibujar las rectas de las restricciones
    for a, b, c, tipo in restricciones:
        if b != 0:
            y = (c - a * x) / b
            plt.plot(x, y, label=f"{a}x + {b}y {tipo} {c}")
        else:
            x_line = np.full_like(x, c / a)
            plt.plot(x_line, x, label=f"{a}x {tipo} {c}")

    # Ejes
    plt.axhline(0, color="black", linewidth=0.8)
    plt.axvline(0, color="black", linewidth=0.8)

    # Encontrar puntos de intersección
    puntos = []
    for (a1, b1, c1, _), (a2, b2, c2, _) in combinations(restricciones, 2):
        det = a1 * b2 - a2 * b1
        if det != 0:  # hay intersección
            x_int = (c1 * b2 - c2 * b1) / det
            y_int = (a1 * c2 - a2 * c1) / det
            if xmin <= x_int <= xmax and ymin <= y_int <= ymax:
                puntos.append((x_int, y_int))

    # Agregar esquinas del gráfico
    puntos += [(xmin, ymin), (xmin, ymax), (xmax, ymin), (xmax, ymax)]

    # Filtrar puntos factibles
    factibles = []
    for px, py in puntos:
        valido = True
        for a, b, c, tipo in restricciones:
            if tipo == "<=" and not (a * px + b * py <= c + 1e-6):
                valido = False
            if tipo == ">=" and not (a * px + b * py >= c - 1e-6):
                valido = False
        if valido:
            factibles.append((px, py))

    # Región factible
    if factibles:
        factibles = np.array(factibles)
        hull = ConvexHull(factibles)
        pts = factibles[hull.vertices]
        plt.fill(pts[:, 0], pts[:, 1], "skyblue", alpha=0.4, label="Región factible")

    # Configuración final
    plt.xlim(xmin, xmax)
    plt.ylim(ymin, ymax)
    plt.xlabel("x")
    plt.ylabel("y")
    plt.legend()
    plt.grid(True, linestyle="--", alpha=0.6)
    plt.title("Restricciones y Región Factible")
    plt.show()


# Ejemplo de uso:
restricciones = [
    (1, 1, 15, "<="),
    (2, 1, 18, "<="),
    (1, 2, 20, "<="),
    (1, 0, 0, ">="),
    (0, 1, 0, ">=")
]

graficar_restricciones(restricciones)

Ejemplo de Aplicación

El siguiente ejemplo muestra un problema de programación lineal con 5 restricciones

Restricciones del Ejemplo

  • x + y ≤ 15
  • 2x + y ≤ 18
  • x + 2y ≤ 20
  • x ≥ 0
  • y ≥ 0

Puntos de Intersección

  • (0, 0)
  • (9, 0)
  • (6, 6)
  • (5, 7.5)
  • (0, 10)

¿Qué hace este programa?

1. Grafica cada restricción como una recta
2. Encuentra todos los puntos de intersección
3. Filtra los puntos que cumplen todas las restricciones
4. Dibuja la región factible (polígono convexo)
5. Muestra el gráfico con leyenda y cuadrícula
