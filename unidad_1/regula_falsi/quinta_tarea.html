Método de Regula Falsi

Método de interpolación lineal para encontrar raíces de funciones mediante aproximaciones sucesivas.

Descripción del Método

El método de Regula Falsi (o método de la falsa posición) es un algoritmo iterativo para encontrar raíces de funciones continuas. Combina las características del método de bisección y de la secante, utilizando interpolación lineal para aproximar la raíz.

xᵣ = xᵤ - f(xᵤ) × (xᵤ - xₗ) / (f(xᵤ) - f(xₗ))

Características:

• Requiere que f(a) × f(b) < 0 (cambio de signo)
• Convergencia garantizada para funciones continuas
• Combina seguridad de bisección con velocidad de secante

def regula_falsi(f, a, b, tol=1e-12, max_iter=1000):
    """
    Método de Regula Falsi para encontrar raíces de f(x) = 0
    
    Parámetros:
    f : función a evaluar
    a, b : intervalo inicial [a, b] con f(a)*f(b) < 0
    tol : tolerancia para la convergencia
    max_iter : máximo número de iteraciones
    
    Retorna:
    x : raíz aproximada
    iteraciones : número de iteraciones realizadas
    """
    
    if f(a) * f(b) >= 0:
        raise ValueError("f(a) y f(b) deben tener signos opuestos")
    
    iteraciones = 0
    
    while abs(b - a) > tol and iteraciones < max_iter:
        fa = f(a)
        fb = f(b)
        
        # Fórmula de Regula Falsi
        x = a - fa * (b - a) / (fb - fa)
        fx = f(x)
        
        if fa * fx < 0:
            b = x
        else:
            a = x
            
        iteraciones += 1
    
    return x, iteraciones

# Ejemplo de uso
def funcion_ejemplo(x):
    return x**3 - 2*x - 5

# Intervalo inicial [2, 3] donde hay cambio de signo
raiz, iteraciones = regula_falsi(funcion_ejemplo, 2, 3)

print("Raíz aproximada:", raiz)
print("Iteraciones realizadas:", iteraciones)

¿Cómo funciona el algoritmo?

1. Verifica que f(a) y f(b) tengan signos opuestos
2. Calcula punto de intersección con línea recta
3. Evalúa f(x) en el punto calculado
4. Reemplaza a o b según el signo de f(x)
5. Repite hasta alcanzar la tolerancia
