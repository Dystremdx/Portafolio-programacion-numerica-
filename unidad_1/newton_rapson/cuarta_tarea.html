Método de Newton-Raphson

Implementación del método iterativo para encontrar raíces de ecuaciones no lineales con convergencia cuadrática y alta precisión.

Descripción del Problema

Se requiere un programa en Python que implemente el método de Newton-Raphson para encontrar raíces de ecuaciones no lineales. El sistema debe permitir al usuario ingresar:

  • La función f(x)
  • Su derivada f'(x)
  • Un valor inicial x₀
xₙ₊₁ = xₙ - f(xₙ) / f'(xₙ)

Fórmula iterativa de Newton-Raphson

Criterio de convergencia:

|xₙ₊₁ − xₙ| < 10⁻¹²

Especificaciones

Entrada

  • Función f(x) como expresión Python
  • Derivada f'(x) como expresión
  • Valor inicial x₀

Proceso

  • Iteraciones hasta tolerancia
  • Protección contra divergencia
  • Convergencia cuadrática

Salida

  • Raíz aproximada
  • Número de iteraciones
  • Estado de convergencia

Ejemplo de Aplicación

Ecuación a resolver
f(x) = x² - 10 = 0
Función
f(x) = x**2 - 10
Derivada
f'(x) = 2*x
Punto inicial
x₀ = 3.0

# Método de Newton-Raphson
# Implementación para encontrar raíces de ecuaciones no lineales

# El usuario ingresa la función y su derivada
funcion = input("Ingrese la función f(x): ")       
derivada = input("Ingrese la derivada f'(x): ")    
x0 = float(input("Ingrese el valor inicial x0: "))

tolerancia = 1e-12
iteracion = 0

while True:
    x = x0
    f = eval(funcion)
    df = eval(derivada)
    x1 = x0 - f/df
    iteracion += 1

    if abs(x1 - x0) < tolerancia:
        break

    if iteracion > 1000:
        print("No converge después de muchas iteraciones.")
        break

    x0 = x1

print("Raíz aproximada:", x1)
print("Iteraciones realizadas:", iteracion)

Conclusión

El programa implementa correctamente el método de Newton-Raphson, permitiendo encontrar raíces de ecuaciones no lineales con alta precisión. El usuario puede ingresar cualquier función y su derivada, y el algoritmo determina automáticamente el número de iteraciones necesarias para cumplir con la tolerancia establecida.

El resultado obtenido demuestra la convergencia cuadrática característica del método, donde la precisión se duplica aproximadamente en cada iteración. La implementación incluye protección contra divergencia mediante un límite máximo de iteraciones.

Aplicaciones:

Este método es fundamental en análisis numérico, optimización, física computacional y cualquier disciplina que requiera resolver ecuaciones no lineales con alta precisión.
