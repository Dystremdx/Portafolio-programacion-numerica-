Método del Punto Fijo

Método iterativo para encontrar raíces mediante aproximaciones sucesivas usando la forma x = g(x)

Descripción del Método

El método del Punto Fijo es un algoritmo iterativo para resolver ecuaciones de la forma f(x) = 0. La idea consiste en reescribir la ecuación como x = g(x) y aplicar iteraciones sucesivas comenzando desde un valor inicial x₀ hasta alcanzar la convergencia.

xₙ₊₁ = g(xₙ)

Características:

• Converge si |g'(x)| < 1 en un intervalo que contiene la raíz
• Requiere una buena elección de la función g(x)
• Simple de implementar pero puede divergir si no se cumplen condiciones

import math

# Método del Punto Fijo
# g(x) = expresión ingresada por el usuario

g = input("Ingrese la función g(x): ")
x0 = float(input("Ingrese el valor inicial x0: "))

tolerancia = 1e-12
iteracion = 0

while True:
    x = x0
    x1 = eval(g)
    iteracion += 1
    
    if abs(x1 - x0) < tolerancia:
        break
    
    if iteracion > 1000:
        print("No converge después de muchas iteraciones.")
        break
    
    x0 = x1

print("Raíz aproximada:", x1)
print("Iteraciones realizadas:", iteracion)

¿Cómo funciona el algoritmo?

1. Se parte de un valor inicial x₀
2. Se evalúa x₁ = g(x₀)
3. Se verifica si |x₁ - x₀| < tolerancia
4. Si no converge, se actualiza x₀ = x₁ y se repite
5. El proceso continúa hasta convergencia o máximo de iteraciones
