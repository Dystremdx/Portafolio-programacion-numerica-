Modelo Individual (Agentes Independientes)

Descripci√≥n del Juego
Din√°mica de intercambio entre personas

Reglas del juego:

  • Hay N personas; cada una inicia con 2 dulces aleatorios a, b, c
  • En cada interacci√≥n: prestatario p‚ÇÅ y prestamista p‚ÇÇ (activos)
  • p‚ÇÇ presta 1 dulce a p‚ÇÅ (p‚ÇÅ temporalmente tiene 3)
  • Si p‚ÇÅ tiene {a}, {b}, {c} ‚Üí canjea por chupet√≠n y se retira
  • Devuelve 1 dulce aleatorio a p‚ÇÇ (puede ser diferente)
  • Si no consigue los 3 tipos, devuelve el dulce prestado

C√°lculo autom√°tico de iteraciones:

\[ \text{max\_iter} = \left\lceil \frac{2N}{3} \right\rceil \]

Se toma el n√∫mero total de dulces iniciales (2N) dividido entre 3

Observaci√≥n: En la pr√°ctica, debido a la aleatoriedad, frecuentemente no todos los agentes logran canjear por chupet√≠n. Suele quedar al menos una persona sin chupet√≠n debido a combinaciones y secuencias de pr√©stamos que agotan las posibilidades locales.

# Simulaci√≥n del juego de los dulces (chupetines)
# Din√°mica: cada persona tiene 2 dulces; puede recibir 1 prestado;
# si consigue {a, b, c} canjea y se retira; devuelve 1 dulce aleatorio.

simular_chupetines_doc <- function(N = 20, seed = NULL) {
  if (!is.null(seed)) set.seed(seed)
  tipos <- c("a", "b", "c")
  
  # C√°lculo autom√°tico de iteraciones seg√∫n f√≥rmula
  max_iter <- ceiling((2 * N) / 3)
  
  # Inicializaci√≥n: cada persona con 2 dulces aleatorios
  personas <- replicate(N, sample(tipos, 2, replace = TRUE), simplify = FALSE)
  obtuvo_chupetin <- rep(FALSE, N) # indicador de retiro
  iter <- 0
  
  tiene_tres <- function(x) length(unique(x)) == 3
  
  # Bucle principal: max_iter interacciones
  for (k in 1:max_iter) {
    iter <- iter + 1
    
    # elegir prestatario y prestamista entre activos
    activos <- which(!obtuvo_chupetin)
    if (length(activos) <= 1) break # ya no hay pares posibles
    
    p1 <- sample(activos, 1) # prestatario
    p2 <- sample(setdiff(activos, p1), 1) # prestamista
    
    # si prestamista no tiene dulces, saltar
    if (length(personas[[p2]]) == 0) next
    
    # p2 presta 1 dulce a p1
    dulce_prestado <- sample(personas[[p2]], 1)
    personas[[p1]] <- c(personas[[p1]], dulce_prestado)
    
    # comprobar si p1 ahora tiene los 3 tipos
    if (tiene_tres(personas[[p1]])) {
      obtuvo_chupetin[p1] <- TRUE
      
      # Devuelve UN dulce aleatorio al prestamista
      if (length(personas[[p2]]) > 0) {
        idx <- sample(seq_along(personas[[p2]]), 1)
        personas[[p2]][idx] <- sample(tipos, 1)
      } else {
        personas[[p2]] <- sample(tipos, 1)
      }
      
      # p1 se retira: vector vac√≠o
      personas[[p1]] <- character(0)
    } else {
      # No consigui√≥ completar: devuelve el dulce prestado
      if (length(personas[[p1]]) > 2) {
        personas[[p1]] <- sample(personas[[p1]], 2)
      }
    }
  }
  
  # Resultados
  total_chupetines <- sum(obtuvo_chupetin)
  cat("Simulaci√≥n terminada\n")
  cat("Personas (N):", N, "\n")
  cat("Iteraciones usadas (ceil(2N/3)):", max_iter, "\n")
  cat("Chupetines obtenidos:", total_chupetines, "\n\n")
  cat("√çndices de personas que obtuvieron chupet√≠n:\n")
  print(which(obtuvo_chupetin))
  
  return(list(
    N = N,
    iteraciones = max_iter,
    total_chupetines = total_chupetines,
    quienes = which(obtuvo_chupetin),
    personas_final = personas
  ))
}

Modelo Cooperativo por Grupo

Extensi√≥n Cooperativa
Grupo que comparte todos los dulces

Reglas del modelo cooperativo:

  • Los N integrantes juntan todos sus dulces en una "bolsa com√∫n"
  • Intercambio tipo A: 3 dulces distintos {A,B,C} ‚Üí 1 chupet√≠n
  • Intercambio tipo B: 6 dulces (2 de cada tipo) ‚Üí 2 chupetines + 1 dulce extra
  • Conversi√≥n inversa: 1 chupet√≠n ‚Üí 3 dulces aleatorios (si no hay intercambios posibles)
  • El proceso contin√∫a hasta que chupetines ‚â• N (todos tienen) o max_iter
Ventaja cooperativa: El modelo demuestra que mediante el uso compartido de recursos, los grupos pueden alcanzar estados donde todos sus miembros obtienen un chupet√≠n, aumentando la probabilidad de √©xito colectivo.

# Modelo cooperativo del juego de los dulces
# Un grupo trabaja junto compartiendo todos sus dulces.

simular_grupo_cooperativo <- function(n_personas = 5, max_iter = 200, seed = NULL) {
  if (!is.null(seed)) set.seed(seed)
  tipos <- c("A", "B", "C")
  
  # Dulces iniciales: 2 por persona
  dulces <- sample(tipos, n_personas * 2, replace = TRUE)
  chupetines <- 0
  iter <- 0
  
  cat("Simulaci√≥n cooperativa: grupo de", n_personas, "personas\n\n")
  
  while (iter < max_iter) {
    iter <- iter + 1
    tabla <- table(dulces)
    
    # Intercambio tipo B: 6 dulces (2 de cada tipo)
    if (all(c("A", "B", "C") %in% names(tabla)) &&
        all(tabla[c("A", "B", "C")] >= 2)) {
      # Eliminar 6 dulces (2 de cada tipo)
      quitar <- c(rep("A", 2), rep("B", 2), rep("C", 2))
      for (q in quitar) {
        dulces <- dulces[-match(q, dulces)]
      }
      chupetines <- chupetines + 2
      # Dulce extra aleatorio
      dulces <- c(dulces, sample(tipos, 1))
      
    # Intercambio tipo A: 3 dulces distintos
    } else if (length(unique(dulces)) >= 3) {
      dulces <- dulces[-match(unique(dulces)[1:3], dulces)]
      chupetines <- chupetines + 1
      dulces <- c(dulces, sample(tipos, 1))
      
    # Si no se puede intercambiar pero hay chupetines
    } else if (chupetines > 0) {
      chupetines <- chupetines - 1
      dulces <- c(dulces, sample(tipos, 3, replace = TRUE))
    } else {
      break
    }
    
    # Mostrar progreso cada 10 iteraciones
    if (iter %% 10 == 0 || iter == 1) {
      cat(sprintf("Iter %d -> Dulces: %d | Chupetines: %d\n", 
                  iter, length(dulces), chupetines))
    }
    
    # Condici√≥n de √©xito
    if (chupetines >= n_personas) {
      cat("\nResultado final\n")
      cat("Iteraciones:", iter, "\n")
      cat("Dulces restantes:", length(dulces), "\n")
      cat("Chupetines totales:", chupetines, "\n")
      cat("El grupo logr√≥ que todos tengan su chupet√≠n.\n")
      return(invisible(list(iter = iter, dulces = length(dulces), 
                            chupetines = chupetines)))
    }
  }
  
  cat("\nResultado final\n")
  cat("Iteraciones:", iter, "\n")
  cat("Dulces restantes:", length(dulces), "\n")
  cat("Chupetines totales:", chupetines, "\n")
  cat("El grupo no logr√≥ completar todos los chupetines.\n")
  
  invisible(list(iter = iter, dulces = length(dulces), chupetines = chupetines))
}

Modelo Individual

Modelo Cooperativo
Colaborativo

Recursos compartidos. Mayor probabilidad de que todos obtengan chupet√≠n. Sistema con redistribuci√≥n estoc√°stica.

Tipo de Problema

Ambos modelos son agent-based con aleatoriedad intr√≠nseca. Simulaci√≥n Monte Carlo para estudio de comportamiento.

Conclusi√≥n General: Hemos definido y simulado dos modelos de intercambio de dulces. El modelo individual muestra c√≥mo la competencia por recursos limitados puede dejar participantes sin recompensa, mientras que el modelo cooperativo demuestra que el trabajo en grupo y la redistribuci√≥n de recursos incrementan la probabilidad de √©xito colectivo. Ambos modelos son ejemplos de sistemas estoc√°sticos discretos √∫tiles para estudiar din√°micas de intercambio en diversos contextos.
