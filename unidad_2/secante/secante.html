Método de la Secante

Método iterativo que aproxima la raíz usando la pendiente de la línea secante entre dos puntos

Descripción del Método

El método de la secante es una técnica iterativa para encontrar raíces que utiliza dos puntos iniciales para aproximar la derivada mediante una diferencia finita. Es similar al método de Newton-Raphson pero no requiere calcular la derivada analítica de la función.

xₙ₊₁ = xₙ - f(xₙ) × (xₙ - xₙ₋₁) / (f(xₙ) - f(xₙ₋₁))

Características:

• No requiere calcular derivadas explícitamente
• Converge más rápido que bisección pero más lento que Newton
• Requiere dos puntos iniciales cercanos a la raíz

import math

def f(x):
    return x**2 - 4

x0 = 1
x1 = 3
tolerancia = 1e-6
iteracion = 0
max_iter = 100

print(f"{'Iter':<6}{'x_n-1':<12}{'x_n':<12}{'x_{n+1}':<12}{'Error':<12}")
print("-"*55)

while True:
    iteracion += 1
    f0 = f(x0)
    f1 = f(x1)
    
    # Fórmula de la secante
    x2 = x1 - f1 * (x1 - x0) / (f1 - f0)
    error = abs(x2 - x1)
    
    print(f"{iteracion:<6}{x0:<12.6f}{x1:<12.6f}{x2:<12.6f}{error:<12.6f}")
    
    if error < tolerancia or iteracion >= max_iter:
        break
    
    x0, x1 = x1, x2

print("\nRaíz aproximada:", round(x2, 6))
print("Iteraciones realizadas:", iteracion)

¿Cómo funciona el algoritmo?

1. Se inicia con dos puntos x₀ y x₁
2. Se calcula la pendiente de la secante entre ambos puntos
3. Se encuentra donde la secante cruza el eje x (nuevo x₂)
4. Se actualizan los puntos: x₀ = x₁, x₁ = x₂
5. Se repite hasta alcanzar la tolerancia deseada
