Ejercicio 1: Modificación de la Matriz de Transición

EJERCICIO 1
Impacto de inversión en infraestructura turística - Isla Taquile
Matriz 4×4

Objetivo:

  • Analizar el impacto de mejoras en infraestructura en Taquile
  • Calcular la nueva distribución estacionaria
  • Comparar con el escenario original
  • Evaluar velocidad de convergencia
Matriz Original
\[T_{\text{original}}\]
Matriz Modificada
\[T_{\text{nueva}}\]

Ecuación para distribución estacionaria:

\[(T_{\text{nueva}}^{T})\mathbf{v} = \lambda\mathbf{v}\]

Donde \(\pi\) es el eigenvector normalizado para \(\lambda = 1\)

Resultados Principales

Destino Original (%) Nueva (%) Cambio (p.p.)
Puno Ciudad 39.22 36.55 -2.67

Isla Amantaní 17.25 15.78 -1.47
Conclusión: La inversión en infraestructura incrementa la participación de Taquile en 27.54%, convirtiéndolo en el segundo destino más importante.
Código R - Distribución Estacionaria 
# Función para calcular distribución estacionaria
calcular_distribucion <- function(matriz) {
    eigen_result <- eigen(t(matriz))
    idx_uno <- which.min(abs(eigen_result$values - 1))
    v <- eigen_result$vectors[, idx_uno]
    pi <- Re(v / sum(v))
    pi <- abs(pi) / sum(abs(pi))
    return(list(
        distribution = pi,
        eigenvalues = eigen_result$values
    ))
}

# Matriz modificada para Taquile
T_nueva <- matrix(c(
    0.25, 0.45, 0.20, 0.10,
    0.40, 0.15, 0.35, 0.10,
    0.30, 0.10, 0.40, 0.20,
    0.55, 0.15, 0.10, 0.20
), nrow=4, byrow=TRUE)

# Calcular distribución
resultado <- calcular_distribucion(T_nueva)
pi_nueva <- resultado$distribution
print(round(pi_nueva * 100, 2))

Código R - Simulación Anual 
# Simulación anual completa
simulacion_anual <- function(dias=365) {
    resultados <- matrix(0, nrow=dias+1, ncol=4)
    estado <- c(1, 0, 0, 0)  # Todos en Puno Ciudad
    resultados[1,] <- estado
    
    for (t in 1:dias) {
        # Determinar temporada según día del año
        if (t <= 120) {
            T_actual <- T_alta
        } else if (t <= 240) {
            T_actual <- T_media
        } else {
            T_actual <- T_baja
        }
        
        # Actualizar estado
        estado <- t(T_actual) %*% estado
        resultados[t+1,] <- estado
    }
    
    colnames(resultados) <- c("Puno", "Uros", "Taquile", "Amantaní")
    resultados_df <- as.data.frame(resultados)
    resultados_df$Dia <- 0:dias
    
    return(resultados_df)
}

# Ejecutar simulación
sim_df <- simulacion_anual(365)

# Calcular promedios anuales
promedios_anuales <- colMeans(sim_df[,1:4]) * 100
cat("Promedios anuales de distribución (%):\n")
print(round(promedios_anuales, 1))
Hallazgo clave: El gap espectral \((1 - |\lambda_2|)\) aumenta de 0.5877 (baja) a 0.7013 (alta), indicando una convergencia 19% más rápida en temporada alta. Esto sugiere que el sistema turístico es más dinámico y se ajusta más rápido durante periodos de alta demanda.
