Definición del Gradiente

El gradiente de una función escalar \( f \) es un vector que contiene todas sus derivadas parciales. Para una función \( f: \mathbb{R}^n \rightarrow \mathbb{R} \), el gradiente se define como:

\[ \nabla f(x_1, x_2, \dots, x_n) = \left( \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \dots, \frac{\partial f}{\partial x_n} \right) \]

Este vector apunta en la dirección de mayor incremento de la función. En métodos numéricos, se utiliza en el descenso del gradiente, donde los valores de las variables se actualizan iterativamente:

\[ x_{i+1} = x_i - \eta \nabla f(x_i) \]

donde \( \eta \) es la tasa de aprendizaje o tamaño de paso.

Ejemplo 1: Función de Una Variable

Función: \( f(x) = x^2 \)

Derivada: \( f'(x) = 2x \)
Tasa de aprendizaje: \( \eta = 0.1 \)
Punto inicial: \( x_0 = 5 \)

# ======================================================
# DESCENSO DEL GRADIENTE - f(x) = x²
# ======================================================

# Definimos la función y su derivada
f <- function(x) { x^2 }
f_deriv <- function(x) { 2*x }

# Parámetros del algoritmo
x0 <- 5
eta <- 0.1
n_iter <- 15

# Inicializamos vectores para almacenar resultados
x_vals <- numeric(n_iter)
f_vals <- numeric(n_iter)
deriv_vals <- numeric(n_iter)

# Asignamos valores iniciales
x_vals[1] <- x0
f_vals[1] <- f(x0)
deriv_vals[1] <- f_deriv(x0)

# Iteraciones del descenso de gradiente
for (i in 2:n_iter) {
    x_new <- x_vals[i-1] - eta * f_deriv(x_vals[i-1])
    x_vals[i] <- x_new
    f_vals[i] <- f(x_new)
    deriv_vals[i] <- f_deriv(x_new)
}

# Mostrar resultados en tabla
resultados <- data.frame(
  Iteracion = 1:n_iter,
  x_i = round(x_vals, 3),
  f_x_i = round(f_vals, 3),
  df_dx = round(deriv_vals, 3),
  x_i1 = c(round(x_vals[-1], 3), NA)
)

print(resultados)

# Graficar la trayectoria
plot(x_vals, f_vals, type="l", col="blue", lwd=2,
     xlab="Valores de x", ylab="f(x)",
     main="Descenso del Gradiente - f(x) = x²")
points(x_vals, f_vals, col="red", pch=19, cex=1.2)

Ejemplo 2: Función de Dos Variables

Función: \( f(x, y) = x^2 + y^2 + 3\sin(x) + 2\cos(y) \)

Derivadas parciales:
\( \frac{\partial f}{\partial x} = 2x + 3\cos(x) \)
\( \frac{\partial f}{\partial y} = 2y - 2\sin(y) \)

Vector gradiente:
\( \nabla f(x, y) = (2x + 3\cos(x),\; 2y - 2\sin(y)) \)

Parámetros:
Tasa de aprendizaje: \( \eta = 0.1 \)
Punto inicial: \( (x_0, y_0) = (5, 3) \)
Iteraciones: 25

gradiente_2d.R 
# ======================================================
# DESCENSO DEL GRADIENTE EN 3D CON PLOT3D 
# f(x, y) = x^2 + y^2 + 3*sin(x) + 2*cos(y)
# ======================================================

library(plot3D)

# Definición de la función y su gradiente
f <- function(x, y) { 
  x^2 + y^2 + 3*sin(x) + 2*cos(y) 
}

grad <- function(x, y) {
  df_dx <- 2*x + 3*cos(x)
  df_dy <- 2*y - 2*sin(y)
  c(df_dx, df_dy)
}

# Parámetros del algoritmo
x0 <- 5
y0 <- 3
eta <- 0.1
n_iter <- 25

# Inicialización de vectores
x_vals <- numeric(n_iter)
y_vals <- numeric(n_iter)
f_vals <- numeric(n_iter)
dfx_vals <- numeric(n_iter)
dfy_vals <- numeric(n_iter)

# Valores iniciales
x_vals[1] <- x0
y_vals[1] <- y0
f_vals[1] <- f(x0, y0)
g <- grad(x0, y0)
dfx_vals[1] <- g[1]
dfy_vals[1] <- g[2]

# Iteraciones del descenso de gradiente
for (i in 2:n_iter) {
  g <- grad(x_vals[i-1], y_vals[i-1])
  x_vals[i] <- x_vals[i-1] - eta * g[1]
  y_vals[i] <- y_vals[i-1] - eta * g[2]
  f_vals[i] <- f(x_vals[i], y_vals[i])
  dfx_vals[i] <- g[1]
  dfy_vals[i] <- g[2]
}

# Creación de tabla de resultados
resultados <- data.frame(
  Iteracion = 1:n_iter,
  x_i = round(x_vals, 6),
  y_i = round(y_vals, 6),
  f_x_y = round(f_vals, 6),
  df_dx = round(dfx_vals, 6),
  df_dy = round(dfy_vals, 6)
)

print(resultados)
