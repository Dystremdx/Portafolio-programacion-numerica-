Análisis Modal Estructural

1. Introducción y Motivación
Contexto del Problema

Importancia del Análisis Dinámico

El análisis dinámico de estructuras es fundamental para garantizar la seguridad ante cargas sísmicas, viento y otras excitaciones dinámicas. Los autovalores y autovectores del sistema matricial (K, M) representan:

  • Frecuencias naturales: Frecuencias propias de vibración libre
  • Modos propios: Formas características de deformación
  • Base para análisis sísmico y de resonancia

Objetivos Específicos

  1. Desarrollar modelo matemático de edificio de 3 pisos
  2. Calcular frecuencias naturales y periodos de vibración
  3. Determinar y visualizar modos propios
  4. Interpretar resultados desde perspectiva ingenieril
  5. Proporcionar código reproducible en R
  6. Establecer recomendaciones de diseño

analisis_modal_edificio.R 
# ============================================
# Análisis Modal - Edificio 3 Pisos
# ============================================

# --- 1. DEFINICIÓN DE PARÁMETROS ---
m <- c(1000, 1000, 1000) # Masas (kg)
k1 <- 20000; k2 <- 18000; k3 <- 15000 # Rigideces (N/m)

# --- 2. CONSTRUCCIÓN DE MATRICES ---
M <- diag(m) # Matriz de masa diagonal

# Matriz de rigidez (ensamblaje por entrepiso)
K <- matrix(c(
  k1+k2, -k2, 0,
  -k2, k2+k3, -k3,
  0, -k3, k3
), nrow=3, byrow=TRUE)

cat("Matriz de Masa M:\n")
print(M)
cat("\nMatriz de Rigidez K:\n")
print(K)

# --- 3. PROBLEMA DE AUTOVALORES ---
A <- solve(M) %*% K # M^(-1) * K
eigen_result <- eigen(A)
omega2 <- eigen_result$values # Autovalores = omega^2
phi_raw <- eigen_result$vectors # Autovectores (modos)

# --- 4. ORDENAR POR FRECUENCIA CRECIENTE ---
idx <- order(omega2)
omega2 <- omega2[idx]
phi_raw <- phi_raw[, idx]

# --- 5. CALCULAR PARÁMETROS MODALES ---
omega <- sqrt(omega2) # Pulsación (rad/s)
freq <- omega / (2*pi) # Frecuencia (Hz)
periodo <- 1 / freq # Periodo (s)

# --- 6. NORMALIZACIÓN DE MODOS ---
# Normalizar: máximo valor absoluto = 1
# Forzar primer componente positivo
phi_norm <- phi_raw
for(i in 1:ncol(phi_raw)) {
  col <- phi_raw[,i]
  col <- col / max(abs(col)) # Normalizar
  if(col[1] < 0) col <- -col # Forzar signo
  phi_norm[,i] <- col
}

# --- 7. PRESENTACIÓN DE RESULTADOS ---
cat("\n=== PARÁMETROS MODALES ===\n")
resultados <- data.frame(
  Modo = 1:3,
  Omega2 = round(omega2, 4),
  Omega = round(omega, 4),
  Freq_Hz = round(freq, 4),
  Periodo_s = round(periodo, 4)
)
print(resultados)

cat("\n=== MODOS PROPIOS NORMALIZADOS ===\n")
colnames(phi_norm) <- paste0("Modo_", 1:3)
rownames(phi_norm) <- paste0("Piso_", 1:3)
print(round(phi_norm, 3))

# --- 8. VERIFICACIÓN DE ORTOGONALIDAD ---
cat("\n=== VERIFICACIÓN: phi^T * M * phi ===\n")
ortog_M <- t(phi_norm) %*% M %*% phi_norm
print(round(ortog_M, 2))

cat("\n=== VERIFICACIÓN: phi^T * K * phi ===\n")
ortog_K <- t(phi_norm) %*% K %*% phi_norm
print(round(ortog_K, 2))
4. Interpretación de Modos
Análisis Físico

Modo 1 - Fundamental (f₁ = 0.305 Hz)

Características: Todos los pisos en fase (mismo sentido). Amplitud crece hacia arriba: 0.396 < 0.755 < 1.000. Mayor energía cinética. Modo más importante para sismo.

Interpretación física: El edificio completo se balancea como péndulo invertido. Piso superior tiene máxima amplitud. Periodo largo (3.27 s) indica estructura muy flexible.

Modo 2 (f₂ = 0.813 Hz)

Características: Piso 1: movimiento positivo (1.000). Piso 2: movimiento positivo reducido (0.662). Piso 3: movimiento negativo (-0.895). Cambio de fase entre pisos 2 y 3. Punto de inflexión entre pisos 2 y 3.

Modo 3 (f₃ = 1.194 Hz)

Características: Patrón: (+) → (-) → (+). Dos cambios de fase. Mayor frecuencia = más "local". Menor participación en respuesta sísmica pero importante para diseño de elementos locales.

Zona Sísmica y Espectro de Diseño

  • Zona sísmica 3 (Lima, Puno): Alta sismicidad
  • Periodo de esquina (Tₐ): ≈ 0.6 s
  • Periodo largo (Tᴸ): ≈ 2.0 s
  • Implicación: T₁ = 3.27 s > Tᴸ → zona de desplazamientos constantes en espectro
6. Medidas de Mitigación
Recomendaciones de Diseño

Corto Plazo

  • Evaluación estructural detallada
  • Modelado 3D con masa distribuida
  • Inclusión de amortiguamiento (ξ = 5%)
  • Verificación de derivas de entrepiso

Mediano Plazo

  • Reforzar columnas de piso 1 (eliminar soft-story)
  • Instalar disipadores de energía
  • Añadir muros de corte (shear walls)
  • Arriostramientos diagonales (bracings)

Largo Plazo

  • Monitoreo sísmico continuo
  • Actualización de modelo con datos reales
  • Plan de mantenimiento preventivo
  • Optimización de ubicación de disipadores
Conclusión General: El análisis modal revela que la estructura tiene un periodo fundamental muy largo (3.27 s), indicando flexibilidad excesiva. Se requieren medidas de reforzamiento inmediatas para cumplir con el código sísmico y garantizar la seguridad estructural ante eventos sísmicos.

Conclusiones Principales

  1. El edificio presenta periodo fundamental muy largo (T₁ = 3.27 s), indicando estructura muy flexible
  2. La frecuencia fundamental baja (0.305 Hz) implica riesgo de resonancia con excitaciones de baja frecuencia
  3. Los modos superiores muestran deformaciones locales importantes para diseño detallado
  4. Se requiere reforzamiento estructural para cumplir códigos sísmicos
  5. El método de autovalores es eficiente y preciso para análisis preliminar

Extensión Académica
