Método de Bisección

Método robusto de búsqueda de raíces que divide el intervalo en mitades sucesivas

Descripción del Método

El método de bisección es uno de los algoritmos más simples y confiables para encontrar raíces de funciones continuas. Se basa en el teorema del valor intermedio: si f(a) y f(b) tienen signos opuestos, entonces existe al menos una raíz en el intervalo [a, b].

m = (a + b) / 2

Características:

• Requiere que f(a) × f(b) < 0 (cambio de signo)
• Convergencia garantizada aunque lenta
• Reduce el error a la mitad en cada iteración

import math

# Definición de la función
def f(x):
    return math.exp(3*x) - 4

a = 0
b = 1
tolerancia = 0.1

print(f"{'Iter':<6}{'a':<10}{'b':<10}{'m':<10}{'f(a)':<12}{'f(b)':<12}{'f(m)':<12}{'Error':<10}")
print("-"*75)

iteracion = 0

while True:
    iteracion += 1
    m = (a + b) / 2
    fa = f(a)
    fb = f(b)
    fm = f(m)
    error = (b - a) / 2
    
    print(f"{iteracion:<6}{a:<10.5f}{b:<10.5f}{m:<10.5f}{fa:<12.5f}{fb:<12.5f}{fm:<12.5f}{error:<10.5f}")
    
    if abs(error) < tolerancia:
        break
    
    if fa * fm < 0:
        b = m
    else:
        a = m

print("\nRaíz aproximada: x =", round(m, 4))
print("Error final:", round(error, 5))

¿Cómo funciona el algoritmo?

1. Verifica que f(a) y f(b) tengan signos opuestos
2. Calcula el punto medio m = (a + b) / 2
3. Evalúa f(m) y determina en qué mitad está la raíz
4. Reemplaza a o b con m según el signo
5. Repite hasta que el error sea menor que la tolerancia
